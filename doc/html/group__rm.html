int opennsl_rm_profile_ports_update ( uint16  client_id,
int  unit,
opennsl_pbmp_t  ports 
)

Used to update the registered ports with the Resource Manager for a given client.

Description
Each client which wants to use the resource manager(RM) functionality in OpenNSL, will register its requirements via a profile structure. If the client later wants to change its ports requirement, it can use this API to update the ports value of the profile.
Parameters:
client_id[IN] client ID returned by RM on successful registration
unit[IN] unit ID for which the ports ownership is to be updated
ports[OUT] New bitmap of the requested ports
Return values:
OPENNSL_RM_E_XXX

int opennsl_rm_register ( opennsl_rm_profile_t *  profile,
uint16 *  client_id 
)

Used to register with Resource Manager.

Description
Each client which wants to use the resource manager(RM) functionality in OpenNSL, should register its requirements via a profile structure. This API validates the passed requirements along with the already registered clients and on successful registration returns a Client ID.
Parameters:
profile[IN] Pointer to the structure filled with resource requirements.
client_id[OUT] Client ID assigned on successful registration
Return values:
OPENNSL_RM_E_XXX
