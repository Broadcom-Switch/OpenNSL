183 /*
184  The Group Table contains one entry for each Group. The table is indexed
185  by the groupId which identifies the group entry. Data is encoded into the groupId to specify the OF-DPA
186  group entry type and information required by OF-DPA to configure the datapath.
187 
188  The groupId encoding method is:
189 
190  L2 Interface, L2 Rewrite types:
191  (MSB to LSB) 4 bits encode the Group Table Entry type |12 bits of VLAN ID | 16 bits of port identifier
192 
193  L2 Multicast, L2 Flood, L3 Multicast, and L3 Interface types:
194  (MSB to LSB) 4 bits encode the Group Table Entry type |12 bits of VLAN ID | 16 bits of index
195 
196  L3 Unicast and L3 ECMP types:
197  (MSB to LSB) 4 bits encode the Group Table Entry type | 28 bits of index
198 
199 */
200
