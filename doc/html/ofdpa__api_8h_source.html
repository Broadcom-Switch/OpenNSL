00191 /*
00192    The Group Table contains one entry for each Group.  The table is indexed
00193    by the groupId which identifies the group entry.  Data is encoded into the groupId to specify the OF-DPA
00194    group entry type and information required by OF-DPA to configure the datapath.
00195 
00196    The groupId encoding method is:
00197 
00198    L2 Interface, L2 Rewrite types:
00199         (MSB to LSB) 4 bits encode the Group Table Entry type |12 bits of VLAN ID | 16 bits of port identifier
00200 
00201    L2 Multicast, L2 Flood, L3 Multicast, and L3 Interface types:
00202         (MSB to LSB) 4 bits encode the Group Table Entry type |12 bits of VLAN ID | 16 bits of index
00203 
00204    L3 Unicast and L3 ECMP types:
00205         (MSB to LSB) 4 bits encode the Group Table Entry type | 28 bits of index
00206 
00207 */
00208
